Si l'identification s'est bien déroulée, un message d'accueil du type apparaîtra :

      Bienvenue "Prénom" "Nom" sur le service de partage de DVDs.

De plus, les listes des DVDs en cours d'emprunt ainsi que ceux réservés seront affichées (cf. partie "Mes DVDs"² pour plus de détails).
En cas d'échec un message d'erreur apparaît :
  • Si le champ 'Login' n'est pas rempli:
          Identification annulée, le login n'est pas correct.
  • Si le champ 'Password' n'est pas rempli:
          Identification annulée, le mot de passe n'est pas correct.
  • Si le login et/ou mot de passe sont incorrects:
          Identification annulée, le nom d'utilisateur n'existe pas ou il ne correspond pas au mot de passe.
  • Si l'erreur provient de l'application:
          Identification annulée, une erreur est survenue: + le détail de l'erreur.

Se déconnecter


Si la déconnection s'est bien déroulée, un message du type suivant apparaîtra :

      A bientôt "Prénom" "Nom".

En cas d'échec un message d'erreur apparaît :
  • Si la sesson était déjà fermée ou si la session à expirée:
          Fermeture de session annulée, auncune session ouverte.
  • Si l'erreur provient de l'application:
          Fermeture de session annulée, une erreur est survenue: + le détail de l'erreur.

Rechercher un DVD


Titre Catégorie Date de parution
(yyyy-mm-dd)		 Actions possibles

Par défaut (ou si les champs ne sont pas remplis), cette page affiche tous les DVDs disponibles.
Afin de rechercher un, ou plusieurs, DVDs en particulier, plusieurs critères de selection sont mis à votre disposition:
  • Le titre du DVD
  • La catégorie du DVD
  • La date de parution du DVD

ATTENTION: Le champ titre est sensible à la casse.

S'il n'existe aucun DVD, le message suivant apparaît :

      Aucun dvd n'a été trouvé.

L'exemple suivant montre les possibilités de navigation à partir du résultat de la recherche:
Titre Catégorie Date de parution
(yyyy-mm-dd)		 Actions possibles
The Matrix Science-fiction 1999-06-23
The Matrix Reloaded Science-fiction 2003-05-16
The Matrix Revolutions Science-fiction 2003-11-05

Vous pouvez interagir avec chaque DVD de différentes manières:
  • Le DVD "The Matrix" est libre, vous pouvez donc l'emprunter;
  • Le DVD "The Matrix Reloaded" est déjà en cours d'emprunt, vous pouvez donc le réserver;
    Il sera automatiquement emprunté pour vous dès qu'il aura été rendu par son emprunteur actuel.
  • Le DVD "The Matrix Revolutions" est est déjà en cours d'emprunt mais est aussi déjà réservé. Vous ne pouvez qu'afficher les détails de ce DVD;
Dans tous les cas, vous pourrez toujours, au moins, afficher les détails d'un DVD.

Ici l'utilisateur a emprunté le DVD "The Matrix" le 18 décembre. Il peut donc soit rendre ce DVD (avant le 1er janvier) soit afficher les détails de ce DVD.

Mes Réservations


DVD Date théorique d'emprunt Actions possibles
The Matrix Revolutions 2007-01-01 Détails
New York 1997

Ici l'utilisateur a réservé le DVD "The Matrix Revolutions". Il lui sera, si l'emprunteur le rend dans les temps, automatiquement emprunté le 1er janvier. Sur ce DVD, l'utilisateur peut annuler sa réservation (avant qu'il soit automatiquement emprunté: le 1er janvier) ou afficher ses détails.

La date théorique d'emprunt du DVD réservé "New York 1997" est en vert car nous sommes à une date supérieure ou égale à cette date. Ce qui veut dire que ce DVD ne devrait pas tarder à passer dans les emprunts de l'utilisateur.

Emprunter un DVD


Pour emprunter un DVD, il suffit d'aller sur la page "Rechercher un DVD" et de cliquer sur le lien "Emprunter" dans les actions possibles du DVD à emprunter.

ATTENTION: Il est impossible d'emprunter un DVD dans les cas suivants:
  • s'il est déjà en cours d'emprunt par vous ou un autre utilisateur;
  • si vous l'avez emprunté et déjà rendu aujourd'hui.
  • si vous avez dépassé votre quota de 3 DVDs empruntés.

Réserver un DVD


Pour réserver un DVD, il suffit d'aller sur la page "Rechercher un DVD" et de cliquer sur le lien "Réserver" dans les actions possibles du DVD à réserver.

ATTENTION: Il est impossible de réserver un DVD dans les cas suivants:
  • s'il n'est pas actuellement emprunté;
  • s'il est déjà réservé par vous ou un autre utilisateur;
  • s'il est actuellement emprunté par vous;
  • si vous avez dépassé votre quota de 3 DVDs réservés.

Afficher les détails d'un DVD


Pour afficher les détails d'un DVD, il suffit d'aller sur la page "Rechercher un DVD" et de cliquer sur le lien "Détails" dans les actions possibles du DVD dont on veut voir les détails.

Remarque: Cette action peut également être effectuée sur les DVDs empruntés ou réservés par l'utilisateur dans la page "Mes DVDs".

Commenter un DVD


Pour donner un avis sur un DVD, il suffit d'aller sur la page de détails du DVD à commenter et de cliquer sur le lien "Ajouter un commentaire". Une seconde page vous permettra de composer votre avis et de la sauvegarder.

Si l'utilisateur a déjà commenté un DVD, il peut soit le modifier soit le supprimer.
